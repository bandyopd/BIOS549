Air pollution data

This dataset contains annual average ambient air pollution data in the US each year from 2005-2014. The data are downloaded from the EPA's Air Quality System (AQS)

https://www.epa.gov/aqs

There are 11 pollutants including both gases and particules: CO, EC, NH4, NO2, NO3, NOx, O3, OC, PM10, PM25, SO2 and SO4. The pollutants are not measured at the same locations. The plots below use only fine (PM2.5) and course (PM10) particulate matter in 2014.

Plotting the data

library(ggplot2)
library(maps)
library(viridis)

hist(PM25,breaks=25)


hist(PM10,breaks=25)


PM10[PM10>100]<-NA # Remove an outlier
hist(PM10,breaks=25)


pm <- data.frame(long=s[,1],lat=s[,2],PM25=PM25)
ggplot(pm, aes(long, lat)) +
  borders("state") +
  geom_point(aes(colour = PM25)) +
  scale_colour_gradientn(colours = viridis(10)) +
  coord_map(projection = "albers", lat0 = 39, lat1 = 45) +
  xlab("")+ylab("")+labs(title="Fine particulate matter, 2014")


pm <- data.frame(long=s[,1],lat=s[,2],PM10=PM10)
